ls -> lista arquivos
cd -> move entre diretórios
pwd -> mostra diretório de trabalho
mkdir -> cria uma pasta
cat -> mostra no git bash o conteúdo de um arquivo de texto
git init . - transforma uma pasta em um repositório git
git commit -m 

FLUXO DE TRABALHO

1. criar diretorio do projeto
2. iniciar o git dentro do diretorio (git init .)
3. edita ou cria arquivos
4. git status
5. git add "nome do arquivo"
6. git status
7. git commit -m "mensagem"
8. git status
9. volta ao passo 3
